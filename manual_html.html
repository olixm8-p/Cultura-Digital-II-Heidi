Manual de HTML

HTML (HyperText Markup Language) es el lenguaje principal utilizado para crear páginas web. Es un lenguaje de marcado que estructura el contenido mediante etiquetas. Cada etiqueta indica al navegador cómo mostrar los elementos (texto, imágenes, enlaces, etc.).

Etiquetas HTML básicas

Etiqueta Descripción Ejemplo
<h1> a <h6> Títulos (de mayor a menor importancia) <h1>Hola Mundo</h1>
<p> Párrafo <p>Texto aquí</p>
<a> Hipervínculo <a href="url">Enlace</a>
<img> Imagen <img src="img.jpg">
<div> Contenedor genérico <div>Contenido</div>

Ejemplo de estructura básica

<!DOCTYPE html>
<html>
<head>
    <title>Mi primera página</title>
</head>
<body>
    <h1>¡Hola Mundo!</h1>
    <p>Este es un párrafo.</p>
</body>
</html>

Consejos para programar en HTML

  • Usa indentación para que el código sea más legible.
  • Verifica que todas tus etiquetas se abran y cierren correctamente.
  • Organiza tu contenido en secciones lógicas (<header>, <main>, <footer>).
  • Combina HTML con CSS para estilos y con JavaScript para funcionalidades.
